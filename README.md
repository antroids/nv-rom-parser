# NV-ROM-PARSER

Nv Rom Parser is a tool to extract some information about Nvidia firmware.

## Installation

Clone repository.

Install rust
```bash
curl --proto '=https' --tlsv1.2 -sSf https://sh.rustup.rs | sh
```

Run
```bash
cargo run -- --help
```
